There are 7 steps to a selenium script. The first 5 steps are the core of your selenium script. The last 2 steps are for reporting and cleanup purposes.    

1. Instantiate a Webdriver object to drive the browser you want to use in your tests
        1. Set location of chromedriver path: System.setProperty("webdriver.chrome.driver", "resources/chromedriver");
        2. WebDriver driver = new ChromeDriver();

2. Navigate to the webpage you want to test
        driver.get("https://www.instagram.com");

3. locate the element on the webpage you need to interact with 
        driver.findElement(By.id("sign-in")); //id value is called sign in 

4. Ensure the browser is in the correct state to interact with that element
        WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));

        WebElement username = wait.until(   // Wait until element is visible, then interact
            ExpectedConditions.visibilityOfElementLocated(By.id("user-name"))//element found using inspect
            //there are many findElement(By. :
            1. (By.name("commit")
            2. By.classname("nav-item") 
            3. By.tagName("a") 
            4. By.cssSelector("a(data-test=sign-in"))
            5. By.xpath, 
            6. By.linkText("Sign In")
            7. By.partialLinkText("Sign")
            8. Id is one of the best locators to use for element identifications
        );

5. perform the action on the element
6. Record the test result
        public TestWatcher watcher = new TestWatcher(){
            @Override
            protected void failed(Throwable e, Description description){
                // Code to take a screenshot when the test fails
                TakesScreenshot ts = (TakesScreenshot) driver;
                File source = ts.getScreenshotAs(OutputType.FILE);
                try {
                    FileUtils.copyFile(source, new File("./Screenshots/" + description.getMethodName() + ".png"));
                } catch (IOException ex) {
                    ex.printStackTrace();
                }
            }
        }

7. Quit the test- it ends the drivers in an instant to stop mutiple windows opening
        driver.quit()
